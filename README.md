# Data-Preparation-and-Processing-Assignment
## Objective
Goal is to do EDA, data preparationa and processing of popular kaggle data sets that need to be cleaned. Detailed data preprocssing cleaning and all the steps - before that do detailed eda -- should be done. Clustering and anomaly elimnations in the data sets should also be performed. Data imputation etc.,. all steps in data preparation including feature processing and feature selection should be done. Furthermore, build various ml models using auto ml including ensemble model.
